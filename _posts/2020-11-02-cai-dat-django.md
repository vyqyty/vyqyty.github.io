Django là một web framework miễn phí mã nguồn mở được viết bằng Python. Trong bài viết này mình dựng server Dev trên Ubuntu 20.04. đã có sẵn Python3.8
```
python3 --version
```
```
Output
Python 3.8.5
```
Cài đặt Django
```
sudo apt install python3-django
```
```
django-admin --version
```
```
Output
3.1.2
```
